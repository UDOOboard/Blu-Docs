Firmware Update (Serial)

In order to update the firmware through the serial connection you need:

  • Texas Instrument SmartRF Flash Programmer 2(SFP2) - at least version 1.7.1
  • Serial converter (3.3V USB/TTL serial converter)

Heads up! The following procedure is for Windows only.

The procedure is the following:

  1. Download the official firmware you want to upload on the UDOO BLU from the Firmware Overview page.

  2. Download and install the official TI SmartRF Flash Programmer 2(SFP2).

  3. Run it.

  4. Connect the 3.3V USB/TTL serial converter to the PC.

  5. In the Connected devices menu on the left, select the entry Unknown of the relative USB Serial Port (COM*)

  6. In the Select Target Device... dropdown menu in the bottom left of the software select CC2650

  7. Connect the 3.3V USB/TTL serial converter to the UDOO BLU like shown the image below.

  8. Power up the UDOO BLU. Make sure to short the D7 pin of the UDOO BLU to GND before powering up the board, this will set the UDOO BLU in boot hardware mode that allows you to update the firmware.

  9. In SFP2 do the following step:

    1. in the Flash Image(s) section select the binary .hex file containing the firmware you want to upload.
    2. in the Actions section tick the Erase, Program and Verify checkboxes.
    3. Run the firmware upload clicking the Play button.

  10. Wait until the end of the uploading procedure. The Status window on the bottom logs the process status. If it ends correctly, the bottom bar becomes green.

  11. Turn off the UDOO BLU, detach the serial converter and turn on the board again to load the firmware just uploaded. You should see the LEDs blinking.
