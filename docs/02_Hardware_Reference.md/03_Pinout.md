In this section you can find the details of the **UDOO BLU** pinout.

**UDOO BLU** Pinout features:
* 8x Digital GPIOs
* 6x Analog Input pins
* Power Pins:
  * 3V pin
  * Vin Pin
  * 2x GNDs
  * Reset Pin
* Other Interfaces:
  * 1x I2C
  * 1x SPI
  * 1x UART Serial

### Digital Pins

You can find the Digital Pins of the UDOO BLU in the image below
